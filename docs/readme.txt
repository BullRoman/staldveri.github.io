Для того чтобы добваить СЛАЙДЕРЫ вам необходимо:
    1. скопировать всё от "<div class="container"..." до закрывающего его "</div>"
    2. изменить цифры id-элементов в самом КОДЕ (#content-1 #flex__nav-1 #burg-svg-1 #flex__menu-1 #charact-1 #buy-1 #sample-1 #call-back-1 #slider-1 #price-1) на соответсвующий (следующий) порядковый номер (в коде рядом с этими элементами стоит символ <!-- # --> )
    3. после того как переписали цифры - переходим в файл "main.js" и находим категорию "слайдеры" (в самом конце)


Для добавление слайдера в список фильтра
    в "data-id" и в "data-category" добавить к примеру "mirror" (с зеркалом) или какой-то другой фильтр



Весь перечнь филььруемых обьектов

    home          - для квартиры
    home-e        - для квартиры - эконом        
    home-i        - для дома - утеплённые
    fire-door     - противопожарные
    technical     - технические
    light-outside - светлые снаружи
    light-inside  - светлые внутри
    mirror        - с зеркалом
    all           - все
